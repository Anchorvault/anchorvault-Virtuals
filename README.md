## 🌟 Overview

Welcome to the **AnchorVault x Virtuals Protocol** frontend platform. AnchorVault represents a paradigm shift in how capital flows globally. By merging the predictability of stablecoin reserves with the decentralization of Virtuals Protocol, we are eliminating the friction, cost, and delays of cross-border settlements. 

This isn't just an upgrade; it's a completely new foundation for real-world assets on-chain.

## 🚀 Key Features

*   **Dynamic Landing Page:** Features a seamless modern UI highlighting the core vision and integration of AnchorVault and Virtuals Protocol.
*   **Interactive Video Hub:** Engaging media display for demonstrations and pitches.
*   **Bento Grid Architecture:** A beautiful, responsive card-based layout to showcase core platform benefits and features.
*   **Waitlist Integration:** Capture early adopter interest directly on the site.
*   **Glassmorphism & Fluid Animations:** Built with Tailwind CSS and Framer Motion for a premium Web3 aesthetic.

## 🛠️ Technology Stack

*   **Frontend Framework:** React 19 + TypeScript
*   **Build Tool:** Vite (Ultra-fast HMR)
*   **Styling:** Tailwind CSS (v4) with custom Glassmorphism components
*   **Animations:** Framer Motion & canvas-confetti
*   **Icons:** Lucide React
*   **Data Visualization:** Recharts

## 💻 Getting Started Locally

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Anchorvault/anchorvault-Virtuals.git
    cd anchorvault-Virtuals
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` to view it in the browser.

4.  **Build for production:**
    ```bash
    npm run build
    ```
